GET /api/recommend

Query params:

ParamTypeDescription
budget_per_week_usdnumberWeekly budget available.
support_available[]array<string>Spouse/partner, Extended family, Carpool friends, None.
transportstringCar | Public transit | Walk/Bike | Rideshare only.
hours_per_week_with_kidnumberTime available.
spousebooleanWhether a spouse/partner is present.
parenting_stylestringhands_off | balanced | hands_on.

child_agenumberPrimary child age for planning.
area_typestringsuburb | rural | urban.
priorities_ranked[]array<string>Ranked: Social, Emotional, Physical, Cognitive.
zipstringPostal code or device location used if omitted.
require_webbooleanForce transparent 502 if < 6 results after fallbacks.
