Live Demo: How It Works 🔄

  1. Complete Intake (budget, transport, schedule, parenting style, priorities, child age).
  2. Real Providers fetched via Google Places near your zip or device location.
  3. AI Curation re‑ranks and explains why each option fits your child and constraints.
  4. Guarantees & Fallbacks ensure at least 6 quality options (or a transparent error if web‑only is required).
  5. Explore & Act with map, contact info, and detail pages; browse “Extraordinary People” with citations.

Architecture 🧩

Mobile App (Expo RN) → Flask API

  • /api/recommend builds real‑business results from Google Places, enriches with Place Details, re‑ranks with AI, and enforces ≥6 results (or 502 with require_web=1).
  • /api/extraordinary-people & /api/deep-research generate profiles, images, and sources with robust JSON parsing.

Caching: AsyncStorage keys (latest_recommendations, last_recommend_query, extraordinary_search_* ).

API Endpoints 📡

GET /api/recommend

Query params:

ParamTypeDescription
budget_per_week_usdnumberWeekly budget available.
support_available[]array<string>Spouse/partner, Extended family, Carpool friends, None.
transportstringCar | Public transit | Walk/Bike | Rideshare only.
hours_per_week_with_kidnumberTime available.
spousebooleanWhether a spouse/partner is present.
parenting_stylestringhands_off | balanced | hands_on.
number_of_kidsnumberTotal children.
child_agenumberPrimary child age for planning.
area_typestringsuburb | rural | urban.
priorities_ranked[]array<string>Ranked: Social, Emotional, Physical, Cognitive.
zipstringPostal code or device location used if omitted.
require_webbooleanForce transparent 502 if < 6 results after fallbacks.

Example Request

GET https://your-backend.example.com/api/recommend?zip=02115&budget_per_week_usd=40&transport=Walk%2FBike&child_age=8&priorities_ranked[]=Physical&priorities_ranked[]=Social&require_web=1

POST /api/extraordinary-people

curl -X POST https://your-backend.example.com/api/extraordinary-people \ 
  -H "Content-Type: application/json" \

Setup & Configuration 🚀

Backend

  1. cd backend
  2. Create .env: 
    GOOGLE_MAPS_API_KEY=...
ANTHROPIC_API_KEY=...
YELP_API_KEY=... # optional
  3. pip install -r requirements.txt
  4. python server.py

Frontend

  1. cd frontend
  2. npm install
  3. npx expo start (press i / a / w)

Troubleshooting & FAQ 🛠️

Maps not rendering (iOS)

Build an Expo Dev Client with native modules and run on simulator/device.

No real providers

Ensure GOOGLE_MAPS_API_KEY is set and pass zip. Try require_web=1 to surface web‑only failures (backend returns 502 if < 6).

Deep research JSON error

Parser auto‑repairs common quote issues; rerun query after clearing cache.

Clear caches

Use in‑app clear (or programmatically remove latest_recommendations, last_recommend_query, and call CacheService.clearCache()).
